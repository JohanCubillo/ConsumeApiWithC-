El proyecto de programación 4 creado por Johan Cubillo.

Se Crea un EXCHANGE de Criptomonedas En C# Mostrando al cliente su informacion y diversos servicios disponibles.

A nivel de Capa presentacion podremos observar varios formularios importantes como Billetera,Perfil,
Comprar Criptomonedas,Registrarse,Logearse,Tradear,Stackear entre otros.

Mediante la arquitectura MVC se conectara a una base de datos en SQL Server y ADO.NET.

Se implementara lo aprendido en clase sobre Herencia,Polimorfismo.
